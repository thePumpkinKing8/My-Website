HTML

Tags

(all tags have "<>" around them.)
  • p: The paragraph tag. Used to display text.
  • h1: The header tag. Displays larger text. h can instead be followed by a 2, 3, 4 and so on to get smaller text.
  • sub: Used to display subscript text.
  • ul: Creates an unordered list.
  • li: used to display a list item inside the ul tag.
  • img: Displays an image. Syntax is as follows. img src="example.jpeg"
  • br: The break tag goes to the next line

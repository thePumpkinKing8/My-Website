making a nav bar

What is a nav bar?

A nav bar, or navigation bar, is used to move to the different pages of your website.

The Code

Step 1

  1. create an unordered list in html
  2. fill the list with the names of your different pages
  3. link each list item to its respective page

Step 2

  1. In css target "ul" and give it a list style type of "none"
  2. set ul's overflow to hidden
  3. target "li" and set it to float: left;
  4. target "li a" and set its displat to block

Step 3

  1. target li and set it to text-decoration:none
  2. target "li a:hover" and set it to have a different background color than the rest of your list
  3. change the padding and margins of your navbar to fit your site and your done
